--------------------- How to run the application? -----------------------

To run : Simply click "run.bat" located at the project root folder "\STARWARS_CZ2002"
To reset data to it's initial state: Simple click at "ResetDB.bat" located at the project root folder "\STARWARS_CZ2002"
